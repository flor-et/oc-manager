# OC Manager
### Last Updated 3/18/2026
## What is an OC?
OC is a shorthand meaning "original character", most commonly used online amongst artists to describe a chraracter they made themselves. An OC can be many things: a persona they use as a self-insert avatar, a character invented to interact with an existing piece of media as a form of fan art, a character for an entirely original story, or just a character, independent of anything. 
## What is the OC Manager?
OC Manager is a project I've created for myself in order to keep track of my own characters, creating a sort of portfolio for each character, keeping notes on them, and whatever else may come along as I start using it.
## Why Create this Project?
I've decided to create this for a few reasons!
  1. Transparently, I'd like to pad out my resume a bit. I have lots of things I've worked on before, but I want to present myself with projects I can really say I'm truly proud of.
  2. I want to work on a project I think I will have motivation to work on and maintain long-term
  3. Similar sites (like Toyhouse) are invite only, preventing large amounts of people (incuding myself) from being able to use the program, and from getting any of the benefits that come along with it
  4. After graduating from university, (May 2026) I worry that I will not have anything to work on, or to keep my mind sharp. With any skill, the rule applies: use it or lose it. I think the education I've received and my ability to write code is a valuable asset, one I don't want to lose.
